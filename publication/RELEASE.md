# OmegA Sovereign
## Publication Front Door

Use this page to choose the correct publication bundle.

## If the audience is public

Use:
- `publication/PUBLIC_RELEASE_BUNDLE.md`
- `publication/OMEGA_MASTER_SYNTHESIS.md` if you need a single umbrella paper

This is the redaction-safe bundle. It is the default choice when:
- you are sharing with the general public
- you want a reviewer-safe package
- you need a publication packet that should not expose private addenda or raw provenance

## If the audience is trusted review / internal

Use:
- `publication/PRIVATE_REVIEW_BUNDLE.md`

This is the broader reviewer-gated bundle. It is the default choice when:
- you want full technical context
- you are preparing critique or internal review
- you need access to supporting drafts, private addenda, or provenance artifacts

## If you are unsure

Start with:
- `publication/PUBLICATION_SET.md`
- `publication/OMEGA_MASTER_SYNTHESIS.md`
- `publication/exports/INDEX.md`

That file is the master map of the corpus and will point you to the right track.

## Recommended rule of thumb

- public audience -> `PUBLIC_RELEASE_BUNDLE.md`
- reviewer or internal audience -> `PRIVATE_REVIEW_BUNDLE.md`
- uncertain -> `PUBLICATION_SET.md`
- single-paper summary -> `OMEGA_MASTER_SYNTHESIS.md`
- export-ready handoff -> `publication/exports/INDEX.md`
